# Rust Projects

Coleção de projetos para prática da linguagem Rust utilizando Cargo Workspaces.

## Pré-requisitos

Certifique-se de ter o `rustup` e o `cargo` instalados: [Guia de Instalação](https://doc.rust-lang.org/book/ch01-01-installation.html#installation)

## Como Executar

Este repositório contém múltiplos projetos. Para rodar um específico a partir da raiz, utilize:

`cargo run -p nome-do-projeto`

Exemplo: `cargo run -p r26-03-2026`

## Comandos Úteis

- Compilar todos os projetos: `cargo build`
- Rodar todos os testes: `cargo test`
- Verificar erros de sintaxe: `cargo check`
